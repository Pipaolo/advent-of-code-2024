# Advent of Code 2024

This repository contains my solutions for the [Advent of Code 2024](https://adventofcode.com/2024) challenges implemented in Go.

## Project Structure

.
├── cmd/
│ └── main.go # Main application entry point
├── internal/
│ └── day_1.go # Solutions for individual days
├── go.mod # Go module definition
└── README.md # This file

## Prerequisites

- Go 1.22.6 or higher

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/yourusername/advent-of-code-2024.git
cd advent-of-code-2024
```

2. Run a specific day:

```bash
go run cmd/main.go
```

## Project Organization

- `cmd/main.go`: Contains the main program entry point
- `internal/`: Contains the implementation of solutions for each day's puzzle
  - Each day's solutions are organized in separate files (e.g., `day_1.go`, `day_2.go`, etc.)
